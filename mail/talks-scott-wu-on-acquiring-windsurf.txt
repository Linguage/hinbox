<h3>一：Cognition 72小时闪电收购Windsurf：AI时代的速度与远见</h3>
<p><strong>引言</strong>
Scott Wu，Cognition的联合创始人兼CEO，在近期的一场访谈中，详细披露了其公司如何在短短72小时内完成对Windsurf的年度收购案。这宗交易不仅是AI领域的一大新闻，更折射出当前技术变革时代下，企业决策、价值判断与执行速度的新范式。本文将深入解析这场“闪电战”的幕后逻辑，以及它对AI行业并购所带来的启示。</p>
<p><strong>核心要点一：机会的捕捉与价值的重估</strong></p>
<ul>
<li><strong>突如其来的契机：</strong> Scott Wu透露，Cognition与外界几乎同时在周五得知Windsurf的潜在拆分信息。迅速识别出Windsurf作为“宝藏”而非“空壳”的潜力，是Cognition能够迅速行动的关键。</li>
<li><strong>Google是否错失良机：</strong> 访谈中提出Google是否“错失了金矿”的问题。Scott Wu的回答暗示，大型企业在组织重构时，确实可能忽略某些“被遗留”的资产的真实价值。他强调，Windsurf拥有出色的产品、完整的客户群、全部代码、专有IP以及一支强大的团队，这些都是其被Cognition看重的核心价值。</li>
<li><strong>互补性是核心驱动：</strong> Cognition专注于核心工程与产品团队，而Windsurf则在市场推广、营销、财务和运营方面具备优势。这种天然的互补性，使得Scott Wu认为这是一次“非常自然的合作”，极大地加速了决策过程。</li>
</ul>
<p><strong>核心要点二：AI时代下的极速并购逻辑</strong></p>
<ul>
<li><strong>“周五晚银行托管”的紧迫感：</strong> Scott Wu将此次收购的急迫性比作“银行在周五晚进入托管，必须在周一给出答案”。他强调，在这种行业巨变且信息高度透明的环境中，客户、团队和整个硅谷都在观望。快速行动是为了稳定军心，向市场传递明确信号，避免价值在观望和等待中流失。</li>
<li><strong>牺牲深度尽调换取速度：</strong> Cognition承认，72小时内无法进行如常规M&#x26;A那般深入的尽职调查。但凭借对AI领域业务的深刻理解和对Windsurf高层（如Jeff、Graham和Kevin）的信任，他们认为在高层次上已能充分理解业务。这表明在特定背景下，战略契合与速度优先于极致的细节核查。</li>
<li><strong>谈判与执行的效率：</strong> 从周五晚的首次接触，到周六口头协议，周日敲定所有法律和商业细节，最终在周一上午完成签署并宣布，整个过程展现出惊人的效率和协调能力。Scott Wu特别赞扬了Windsurf团队为此付出的“不眠不休”的努力，这凸显了交易双方在紧迫关头下的高水平协同。</li>
</ul>
<p><strong>核心要点三：创始人与并购的“新常态”</strong></p>
<ul>
<li><strong>“与船共沉”契约的演变：</strong> Scott Wu提到，传统的“创始人与船共沉”的契约在过去一年有所改变，他对此感到“有点失望”。这暗示了在快速变化且充满不确定性的AI时代，创始人和团队面对的退出或整合路径变得更加多样，而不再是简单的“成功”或“失败”。</li>
<li><strong>对未来并购模式的启示：</strong> 尽管Scott Wu认为现在断言这种极速收购会成为“新常态”还为时过早，但Windsurf案例无疑提供了一个新的模板。它表明，当战略契合度极高、市场窗口期极短时，速度、互信和对核心价值的精准判断将成为决定交易成败的关键因素。</li>
</ul>
<p><strong>结论</strong>
Cognition对Windsurf的闪电收购，不仅仅是一次简单的商业交易，更是AI时代下企业如何捕捉机遇、评估价值、快速决策并高效执行的生动案例。它提醒我们，在技术飞速迭代的今天，市场竞争的节奏已然加快，敏锐的洞察、果敢的行动以及对核心人才与资产的精准把握，将成为企业制胜未来的重要筹码。</p>

<h3>二：重新定义软件工程师：AI Agents如何重塑软件工程的未来</h3>
<p><strong>引言</strong>
在人工智能浪潮席卷全球的当下，软件工程领域正经历着前所未有的变革。Cognition CEO Scott Wu在近期访谈中深入探讨了AI，特别是AI Agents，将如何根本性地重塑工程师的角色与工作方式。他大胆预测，未来软件工程师的效率将实现“10倍”提升，而我们所理解的“代码编写”也将被“意图表达”所取代。</p>
<p><strong>核心要点一：效率的指数级跃升与代码的“生产力悖论”</strong></p>
<ul>
<li><strong>当前效率已显著提升：</strong> Scott Wu指出，目前使用AI工具的工程师效率已达到未使用的工程师的 $1.5$ 到 $2$ 倍。他引用Salesforce和Robinhood高管的观点，即高达 $50%$ 的新代码已由AI生成，尽管他强调需考虑代码的重要性与AI辅助的程度。</li>
<li><strong>未来 $10$ 倍效率的展望：</strong> Scott Wu乐观预测，在未来三年内，工程师效率将达到 $10$ 倍提升。更令人兴奋的是，他认为这种效率的提升将引发“Jevons悖论”——即效率的提高将导致代码总量的 $10$ 倍增长。</li>
<li><strong>“代码饥渴症”：</strong> Scott Wu生动地描述了当前软件的普遍缺陷，从银行APP到医疗系统，再到保险服务，许多软件的用户体验远未达到最佳。他认为，市面上所有软件都可提升 $10$ 倍，且有 $10$ 倍多的新软件需要被编写。AI的赋能将释放巨大的创造力，满足这种未被满足的“代码饥渴”。</li>
</ul>
<p><strong>核心要点二：Agent的终极愿景：从“编写”到“拥有”与“表达”</strong></p>
<ul>
<li><strong>Agent接管“所有权”：</strong> Scott Wu的核心观点是，未来的AI Agent将不仅仅是代码助手，而是真正能够“接管工作的所有权”。这意味着Agent将能独立完成从理解需求到执行、调试和部署的整个软件开发生命周期。</li>
<li><strong>“Tony Stark与Jarvis”：</strong> 他用Tony Stark与Jarvis的交互来描绘未来的理想场景——工程师不再需要打开笔记本电脑编写代码，而是直接通过表达意图来指挥计算机完成复杂任务。代码将从“人机交流的语言”退化为“计算机内部执行的细节”。</li>
<li><strong>角色转型：技术架构师与产品经理：</strong> 在Agent深度参与的未来，工程师的价值将不再体现在编写代码的能力，而是聚焦于更高级别的“决策”。他们将更多地扮演技术架构师、技术产品经理的角色，专注于定义问题、构思解决方案、设计系统架构，以及与Agent进行高层次的沟通与协作。</li>
<li><strong>深层语境是当前挑战：</strong> Scott Wu指出，目前Agent面临的最大挑战是“深层语境”的理解——即在大型复杂代码库中建立直觉、理解各组件交互，并有效利用各种工具进行调试和诊断。这不是沙盒问题，而是高度实践性的挑战。</li>
</ul>
<p><strong>核心要点三：IDE与Agent的融合：人机协作的新范式</strong></p>
<ul>
<li><strong>互补的过渡阶段：</strong> Scott Wu认为，在AI Agent完全成熟之前，IDE（集成开发环境）和Agent将在未来几年内成为开发者工作流中不可或缺的一部分。他预见，IDE与Agent的结合将创造全新的、无缝的“同步到异步再到同步”的工作体验。</li>
<li><strong>理想的工作流程：</strong> 工程师可以在IDE中规划任务，利用AI的智能检索和上下文理解来制定决策；然后将大部分繁重的工作交给Agent异步执行；最后，在IDE中审查代码，并进行必要的本地修改。这种模式将最大化人类与AI的协作效率，让工程师能够专注于创造性与战略性任务。</li>
<li><strong>Devin与Windsurf的融合愿景：</strong> 收购Windsurf后，Cognition将致力于探索如何将Devin（AI Agent）与Windsurf（IDE）的优势结合。目标是为同时使用两者的用户提供极致流畅的体验，实现人机协同的最高境界。</li>
</ul>
<p><strong>结论</strong>
Scott Wu对AI Agents和软件工程未来的展望是激动人心的。它预示着一个工程师将从繁琐的编码工作中解放出来，转而专注于更高价值的创新与架构的时代。这不仅是对效率的简单提升，更是对软件工程师职业定义，乃至人机交互方式的深远重塑。我们正站在一个新时代的门槛上，见证着人类智能与人工智能的边界在代码世界中不断消融。</p>

<h3>三：AI的宏观脉动：人才争夺、价值流向与强化学习的隐形引擎</h3>
<p><strong>引言</strong>
当前，人工智能领域正经历着爆炸式增长，引发了关于人才、价值创造和技术突破的广泛讨论。Cognition CEO Scott Wu在一次深度访谈中，分享了他对AI宏观格局的独特见解，包括对人才战的看法、AI价值在产业链中的分布，以及被他视为“隐形引擎”的强化学习（RL）的深远影响。</p>
<p><strong>核心要点一：人才争夺的本质与AI核心群体</strong></p>
<ul>
<li><strong>“合理”的人才战：</strong> 针对外界普遍认为AI人才争夺“失控”的观点，Scott Wu提出了一个“也许疯狂、但合理”的看法。他认为，鉴于AI是“我们一生中最大的技术变革”，对顶尖人才的投入是完全合理的，甚至仍显不足。</li>
<li><strong>塑造AI未来的“极少数人”：</strong> 他指出，真正决定AI发展轨迹的人非常稀少，数量大约在 $100$ 到 $1000$ 人之间，肯定远少于 $10000$ 人。这个精英群体是全球AI进步的核心驱动力，他们的稀缺性直接导致了激烈的人才竞争。</li>
<li><strong>应用层人才竞争的激烈：</strong> 令人惊讶的是，Scott Wu认为，即使在应用层，人才竞争的激烈程度也可能不亚于基础模型实验室。这表明AI的价值创造正向更广阔的领域渗透，对具备AI应用能力的综合型人才需求旺盛。</li>
</ul>
<p><strong>核心要点二：AI价值的多元化归属与市场格局</strong></p>
<ul>
<li><strong>价值在于“差异化”：</strong> Scott Wu驳斥了“AI价值只归属于芯片、基础模型或应用层”的单一论调。他强调，价值将累积在任何能够建立“真正差异化”的领域。这意味着无论是在基础设施、基础模型还是应用层面，只要能提供独特且不可替代的价值，就能在AI生态中占据一席之地。</li>
<li><strong>应用层与基础模型层的共生关系：</strong> Scott Wu认为，应用层（如Cognition）与基础模型层（如Anthropic）的关系更多是“合作”而非简单的“竞争”。他指出，双方关注的问题不同：基础模型旨在提升通用智能，而应用层则专注于如何将这种智能优化并应用于特定垂直领域（如软件工程）的具体产品体验。</li>
<li><strong>驳斥“泡沫论”与投资前景：</strong> 面对普遍存在的“AI泡沫”论调，Scott Wu表现出极强的信心。他引用Sam Altman十年前对YC公司估值的“泡沫论”反驳，认为AI的价值创造潜力被严重低估。他坚信，无论基础模型公司（如OpenAI、Anthropic）还是应用层公司（如Perplexity、Cognition），其整体估值在未来五年都将大幅增长，两者都是“绝佳的投资”。</li>
<li><strong>基础模型层的整合趋势：</strong> 尽管市场潜力巨大，Scott Wu预测基础模型层最终将走向整合，全球可能只剩下 $2$ 到 $6$ 家（更可能 $3$ 到 $5$ 家）主要玩家。这种整合将是市场效率和资源集中的自然结果。</li>
</ul>
<p><strong>核心要点三：被低估的“隐形引擎”——强化学习（RL）</strong></p>
<ul>
<li><strong>RL：过去一年半的最大突破：</strong> Scott Wu明确指出，强化学习（RL）是过去一年半AI能力上“最大的突破”，其潜力被人们严重低估。他认为，RL使得模型能够“解决任何基准问题”，只要有清晰的成功/失败反馈和明确的环境定义。</li>
<li><strong>从“模仿学习”到“行为优化”：</strong> 他解释道，早期的模型（如GPT-3/3.5）主要基于“模仿学习”，即通过阅读海量互联网数据来模仿人类行为。而RL则更进一步，通过不断试错和奖励机制来优化特定行为，使其在特定任务上超越模仿。</li>
<li><strong>数据范式的转变：质量重于数量：</strong> RL的崛起也带来了数据范式的深刻转变。Scott Wu观察到，AI领域正从追求“海量数据”转向追求“一小部分高度策展的、针对特定用例的”高质量数据。他以Cognition的Kevin-32B模型为例，证明了通过RL在少量高度特定数据上的训练，能让模型在特定任务（如CUDA kernels编写）上取得远超传统SOTA模型的表现。这意味着未来AI的突破将更多依赖于“计算”和“高质量数据”，而非单纯的“大数据”。</li>
</ul>
<p><strong>结论</strong>
Scott Wu的访谈勾勒出AI时代下一幅充满活力与挑战的画卷。他不仅为我们揭示了AI浪潮下企业并购的全新速度与逻辑，更通过对人才、价值和核心技术（特别是强化学习）的深度剖析，展现了AI行业远未被完全理解的巨大潜能。这场变革不仅仅是技术的革新，更是商业模式、人才结构乃至人类工作与生活方式的全面重塑。</p>

<p><strong>Scott Wu:</strong> 是的，绝对会。顺便说一句，人们总是问我这个问题。我这么说的原因是因为，如果您问我，我会给您我的大胆预测，那就是我们一直以来对学校的抱怨是，它们应该教您计算机科学和软件工程，但在您知道的，就像您去学校学习垃圾回收、算法和架构等等，然后您去工作，您很多时候必须做的是，调试JavaScript追踪，对吧？有趣的是，就像我们说的，这种解决问题和从第一性原理思考的核心技能，以及理解计算机模型和理解很多这种架构，比如数据库是如何工作的，这些实际上将是未来重要的东西。所以，您知道，也许一种说法是，归根结底，计算机科学学位更像是一个关于如何思考的学位，这永远都会有价值。</p>
<p><strong>Harry Stebbings:</strong> Scott，我想以一个非常重要的问题结束。我非常享受这次谈话。我看到了一个TikTok视频，它说如果您想与某人建立关系，请问他们最引以为豪的特质是什么，或者他们最喜欢自己的什么特质，因为在未来的互动中，您可以通过提及它让他们感觉良好。我很感兴趣。您最引以为豪的特质是什么？您最喜欢自己的什么特质，为什么？</p>
<p><strong>Scott Wu:</strong> 是的。那很有趣。顺便说一句，我其实也很想听听您对这个问题的答案。但是，是的，我发现我的答案，有趣的是，我实际上会说是某种情绪上的平静。我想，我变得非常……对于……我变得非常……我非常苛刻。我是一个非常有竞争力的人。我很容易沮丧等等。但是，，无论好坏，我们经历了很多紧张的情况，您可以想象。特别是在过去的六天里，我们经历了很多。但老实说，这几乎是公司整个历史的故事。这可能比平均水平多一点，您知道，比平均一周多一点，但差别不大。，是的，我引以为傲的是，我能够在这种情况下保持镇定。</p>
<p><strong>Harry Stebbings:</strong> 伙计，我爱您。您太棒了。真的，您是这个行业里最被低估的个性。您需要做更多。Scott这个品牌需要成为一个更大的个人品牌。</p>
<p><strong>Scott Wu:</strong> 您有什么建议？我把您视为Cognition的临时首席营销官。</p>
<p><strong>Harry Stebbings:</strong> 您的产品有趣的地方在于，您属于耐克一类。我说的耐克一类是什么意思？耐克的成功在于，他们让您感觉像个超级英雄。他们告诉您，每个人都是运动员，即使您没有技能，您也能够做以前从未做过的事情。您和Devin也属于这种类似的“人类增强”类别。所以，您应该更好地讲述这些故事。然后您应该更直白、更清晰地讲述您自己的成长故事。人们渴望成为火箭飞船的一部分，而八卦是恶毒的，当您不自己塑造叙事时，它就会发生。如果您塑造叙事，八卦就不会真的发生，因为我不能说您收入2000万美元，而您不是。您的收入要高得多。所以不要让别人为您塑造叙事。</p>
<p><strong>Scott Wu:</strong> 是的。不，说得对。有一种真正的转变，那就是几个月前，我们觉得最好是人们不知道AI代理正在发挥作用，我们更愿意那样。在过去的几个月里，每个人都在尝试做AI代理，所以我们不如更公开一些。</p>
<p><strong>Harry Stebbings:</strong> 但我也认为仅仅是为了人才，为了资金，人们也想要品牌。</p>
<p><strong>Scott Wu:</strong> 是的。是的。不，这很有道理。</p>

<h2>要点回顾</h2>
<h3>视频简介</h3>
<ul>
<li>Scott Wu 是 Cognition 的联合创始人兼首席执行官，该公司推出了世界上第一个 AI 软件工程师 Devin。</li>
<li>上周五，Cognition 在与 Google 达成许可协议后，完成了本年度的收购案，收购了 Windsurf。</li>
<li>Scott Wu 曾是一名世界级的竞技程序员，获得国际信息学奥林匹克金牌，并且是美国数学和物理奥林匹克队成员。</li>
<li>在 Cognition 之前，他是 Scale AI 的创始工程师，帮助塑造了早期的 AI 基础设施堆栈。</li>
</ul>
<h3>Windsurf 收购案：过程、交易与理由</h3>
<ul>
<li><strong>发现收购机会：</strong> Cognition 在周五与其他人同时得知 Windsurf 分拆，并发现这是一个潜在的收购机会。</li>
<li><strong>初步接触与自然契合：</strong> Windsurf 团队和产品被认为是与 Devin 自然互补的。Windsurf 拥有优秀的市场推广、营销、财务和运营团队，而 Cognition 则专注于核心工程和产品团队。</li>
<li><strong>快速决策与执行：</strong>
<ul>
<li>Cognition 于周五晚间首次冷接触 Windsurf。</li>
<li>双方迅速达成共识，认为交易应尽快完成，最好在周一前准备好并宣布。</li>
<li>这种速度类似于银行进入托管状态后需要在短时间内给出解决方案的情况。</li>
<li>团队认为拖延一个月甚至一周都不合适，因为客户和团队都在等待明确的答案。</li>
</ul>
</li>
<li><strong>Windsurf 的价值：</strong>
<ul>
<li>外界普遍认为 Google 留下了“空壳”，但 Scott Wu 认为 Windsurf 是一个“宝藏”。</li>
<li>Windsurf 拥有出色的产品、完整的客户群、全部代码、大量数据和专有 IP。</li>
<li>Windsurf 还有一支令人难以置信的团队。</li>
<li>Cognition 恰好拥有 Windsurf 所需的互补部分。</li>
</ul>
</li>
<li><strong>交易流程：</strong>
<ul>
<li>周六达成口头协议。</li>
<li>周日敲定所有细节、条款和法律文件。</li>
<li>周一早上签署并宣布。</li>
<li>Scott Wu 特别感谢 Windsurf 的 Jeff、Graham 和 Kevin 及其团队为交易付出的努力。</li>
</ul>
</li>
<li><strong>Windsurf 团队的处境：</strong> Windsurf 团队提前几天得知了与 Google 的许可协议，面对选择：作为独立公司运营并筹集新资金，或者寻找合适的合作伙伴。</li>
</ul>
<h3>Google 是否错过了 Windsurf 团队和 IP 中的“金矿”？</h3>
<ul>
<li>Scott Wu 认为，企业在分拆时确实常常会留下非常有价值的部分。</li>
<li>他认为这桩收购对 Cognition 而言是一笔“史诗级交易”。</li>
<li>关于这种“不同寻常”的交易结构是否会成为新常态，Scott Wu 认为创始人的“与船共沉”契约在过去一年有所改变，这有点令人失望。</li>
</ul>
<h3>塑造 AI 未来的人群</h3>
<ul>
<li><strong>AI 人才战：</strong> Scott Wu 认为 AI 领域的人才战是“相当合理的”，因为 AI 是我们生活中最伟大的技术变革。</li>
<li><strong>AI 的影响力：</strong> 即使 AI 能力停滞不前，它也已然是比互联网更大的技术变革。</li>
<li><strong>关键人才数量：</strong> 真正决定 AI 发展轨迹的人非常少，大约在 $100$ 到 $1000$ 人之间，肯定少于 $10000$ 人。</li>
</ul>
<h3>应用层能与模型巨头竞争吗？</h3>
<ul>
<li><strong>对基础模型的依赖：</strong> 像 Windsurf 曾非常依赖 Anthropic，这引发了应用层公司对基础模型的依赖问题。</li>
<li><strong>价值的产生：</strong> Scott Wu 认为 AI 领域的价值产生于能够建立真正差异化的任何领域。</li>
<li><strong>Cognition 的差异化：</strong> Cognition 专注于优化软件工程中的特定能力，并提供围绕这些能力的优秀产品体验。</li>
<li><strong>基础模型的商品化：</strong> Scott Wu 认为基础模型层自然会有竞争，最终会走向一定程度的商品化，这有利于应用层获得杠杆。</li>
<li><strong>合作而非所有：</strong> 基础模型公司之所以与应用层合作，是因为双方关注的问题非常不同。应用层专注于“人类和 AI 如何协同工作来生产代码”，而不仅仅是构建更智能的基础模型。</li>
<li><strong>AI 编码中的深层语境：</strong> AI 编码的关键挑战在于处理“深层语境”，即理解大型复杂代码库、各部分如何交互，以及利用各种工具进行调试和诊断。这不是在沙盒中可以解决的问题。</li>
</ul>
<h3>AI 编写代码的未来及开发者效率</h3>
<ul>
<li><strong>AI 编写代码的比例：</strong> Salesforce 和 Robinhood 等公司表示其 $50%$ 的新代码由 AI 创建。Scott Wu 认同这个趋势，但强调应该考虑每行代码的重要性以及 AI 辅助的程度。</li>
<li><strong>工程师效率提升：</strong> 目前，使用 AI 工具的工程师效率大约是未使用 AI 的 $1.5$ 到 $2$ 倍。</li>
<li><strong>未来展望：</strong> Scott Wu 预计三年内，工程师效率将达到 $10$ 倍提升，并且会有 $10$ 倍以上的代码量被编写。</li>
<li><strong>软件的现状与未来：</strong> 现有的软件有很多不足之处，用户对此已习以为常。AI 将使所有软件提升 $10$ 倍，并创造 $10$ 倍以上的代码需求。</li>
<li><strong>Agent 的未来：</strong> Scott Wu 认为 Agent 最终将能够真正“接管”工作的所有权。未来的工程师将更多地是技术架构师或技术产品经理，专注于决定构建什么解决方案以及如何架构，而不是编写具体代码。编程将变成表达意图，计算机将直接执行。</li>
<li><strong>Devin 的定价：</strong> Devin 的收费是基于使用量按小时计费，旨在比工程师时间价值便宜约 $10$ 倍。例如，考虑到软件工程师年薪约 $30$ 万美元，Devin 旨在提供 $0.5$ 倍的额外价值。</li>
<li><strong>AI 价值累积：</strong> 全球有 $3000$ 万软件工程师，AI 将使他们效率提高 $10$ 倍，并编写 $10$ 倍以上的代码。AI 公司将从中获得可观的价值。</li>
</ul>
<h3>Cognition 的增长与营销</h3>
<ul>
<li><strong>Devin 的增长：</strong> 在过去 $6$ 个月（1 月至今），Devin 的使用量增长了约 $5$ 到 $10$ 倍，包括自助服务和企业客户。</li>
<li><strong>收入增长：</strong> 即使不包括 Windsurf 的交易，Cognition 在过去 $6$ 到 $8$ 个月内也实现了从约 $0$ 到 $8000$ 万美元的年化经常性收入（ARR）增长。</li>
<li><strong>营销策略：</strong> Devin 的增长主要来自真实的工程团队在 Slack 和 Linear 中标记 Devin，并在 GitHub 中使用其拉取请求，它专注于为工程师提供加速工作流程的工具，而不是面向普通消费者。</li>
<li><strong>未来的营销：</strong> Scott Wu 承认 Cognition 在销售和营销方面可以做得更好，现在收购了 Windsurf 的营销团队，希望有所改善。</li>
<li><strong>市场格局：</strong> Scott Wu 认为现在谈论市场格局还为时过早，因为软件工程的未来远比现在设想的要先进。</li>
</ul>
<h3>IDEs 与 Agents：未来的训练轮？</h3>
<ul>
<li><strong>人机界面的演进：</strong> IDEs 和编码 Agent 是下一代人机界面的开端。最终，用户将通过表达意图来命令计算机，而不是通过代码。</li>
<li><strong>Tony Stark 的例子：</strong> Scott Wu 举例 Tony Stark 不需要打开笔记本电脑，而是与 Jarvis 对话来完成工作，预示了未来直观的人机交互。</li>
<li><strong>未来展望：</strong> 这一演变需要几年时间，但每隔 $2$ 到 $3$ 个月就会有新的产品体验进步。</li>
</ul>
<h3>竞争与独特方法</h3>
<ul>
<li><strong>竞争对手：</strong> Scott Wu 认为 NVIDIA 的 Jensen Huang 曾说过，“当你找到公司获胜的方式，意味着没有人需要失败”，这表明每个公司都有其独特的利基市场。</li>
<li><strong>Cognition 的独特性：</strong> Cognition 始终拥有独特的方法和对未来愿景的看法，这使其有别于其他公司。</li>
<li><strong>多个赢家：</strong> Scott Wu 坚信 AI 编码领域会有多个赢家。</li>
</ul>
<h3>未建造的渴望与整合体验</h3>
<ul>
<li><strong>整合 IDE 和 Agent 体验：</strong> Cognition 最想构建的是 IDE 和 Agent 的结合体验。</li>
<li><strong>未来工作流：</strong> 开发者可以在 IDE 中规划任务，利用 AI 的智能和信息检索功能，然后将大部分工作交给 Agent 完成，最后在 IDE 中审核和修改代码。</li>
<li><strong>同步与异步：</strong> 这种结合将实现同步与异步工作的无缝切换，让开发者在需要时介入，并在不需要时让 Agent 并行处理。</li>
<li><strong>Windsurf 和 Devin 的结合：</strong> 在短期内，将保持 Windsurf 和 Devin 产品的各自理念，但会探索如何为同时使用两者的用户提供平滑的切换体验。</li>
</ul>
<h3>快问快答</h3>
<ul>
<li><strong>睡眠情况：</strong> Scott Wu 承认过去一周因交易和团队整合而睡眠不足。</li>
<li><strong>一个被广泛持有的 AI 错误信念：</strong> 人们低估了强化学习（RL）的力量和潜力。
<ul>
<li><strong>RL 的进步：</strong> 过去一年半，RL 是 AI 能力上的最大突破。它能解决几乎所有基准问题，只要能清晰定义所需的行为、操作环境以及成功或失败的条件。</li>
<li><strong>数据的重要性转变：</strong> AI 领域在 $24$ 个月前关注的是海量数据，现在则更多关注一小部分高度策展的、针对特定用例的数据。例如，通过 RL 在 CUDA kernel 代理轨迹上的训练，可以显著提高模型在相关任务上的表现。</li>
<li><strong>结论：</strong> 更多的是计算（compute），而不是海量数据；数据质量重于数量。</li>
</ul>
</li>
<li><strong>对新学生学习计算机科学的建议：</strong> 绝对应该学习 CS。CS 不仅仅是学习具体的技术，更是学习如何从第一性原理思考问题，以及理解计算机模型和架构（如数据库工作原理）。这些是永远有价值的技能。</li>
<li><strong>最引以为傲的个人特质：</strong> 情绪上的冷静。尽管他本人好胜且容易沮丧，但在压力情境下（尤其是在公司和近期交易中）能够保持镇定。</li>
<li><strong>营销建议：</strong> 像 Nike 一样，让用户感觉自己是“超级英雄”，通过人类增强的故事来营销。同时，更直接地分享 Cognition 自己的成长故事，主动塑造叙事，避免流言蜚语。Scott Wu 承认，过去公司更倾向于低调，但现在大家都知道 Agent 技术在发展，应该更加公开。</li>
</ul>
